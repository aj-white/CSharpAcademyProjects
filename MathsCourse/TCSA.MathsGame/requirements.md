# Maths Game

## Requirements

- Create a game that asks user the result of a math questions (e.g. 9 x 9 = ?)
- Collect the input and add a point if answer correct
- A game round has at least 5 questions
- Division should result on INTEGERS only and dividend should go from 0 to 100.
- App should not present questions that result in a non-integer e.g. 7 / 2
- Users should be presented with a menu to choose an operation (add, substract, multipl etc)
- Record previous games in a List and an option in menu should be to get history of previous games
- No need to save to a database, once the App is closed results will be deleted.

### Optional challenges

- Implement levels of difficulty
- Add a timer to track how long user takes to finish game
- Create random game option user gets presented with random operations
