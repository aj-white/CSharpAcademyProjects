# CSharp Academy Projects

## 1. Maths Project

Requirements: [CSharpAcademyMathsProject](https://www.thecsharpacademy.com/project/53/math-game)

This is a simple maths games that asks the user to select a maths operation (Add, Subtract, Multiply, Divide) and then asks 5 questions using that operator. The game hsitory is stored while the app is running and can be viewed by the user.

This project includes several of the additional challenges, adding a mixed operator round and difficulty ratings.
The difficulty ranges can be configured in the code before running.

Learnt skills:
- Builder pattern for a class
- Basic dependency injection
